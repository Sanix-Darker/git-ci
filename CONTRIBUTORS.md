## CONTRIBUTING

Thank you for your interest in contributing to this project! Before you start, please make sure to discuss the
changes you wish to make by creating an issue.

### HOW TO CONTRIBUTE

1. Create an issue to discuss the changes you want to make.
2. Once the issue is approved, fork the repository.
3. Make your changes in a new branch.
4. Test your changes.
5. Create a pull request referencing the initial issue.

### CODE STYLE

• Follow the existing code style.
• Use meaningful variable and function names.
• Write clear commit messages.

### TESTING

• Ensure that your changes do not break existing functionality.
• Write tests for new functionality.

Thank you for your contribution!
